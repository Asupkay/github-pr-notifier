Login

For the extension to retrieve your pull requests, you'll need a GitHub personal access token, which you can create here. When setting it up, choose an expiration date. If you set one, you'll need to log in again once it expires. Selecting 'No expiration' allows continuous access without needing to re-authenticate.

You'll also need to select repository access: public repositories, all repositories, or selected repositories. If you choose 'All' or 'Selected', be sure to grant read-only access to Pull Requests.
